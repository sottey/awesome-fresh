# Fresh Plugins

This directory contains **Fresh editor plugins** and plugin-related resources.

Fresh plugins are written in **TypeScript** and run in a **sandboxed Deno environment**.  
They extend Fresh with commands, UI helpers, editor behavior, and integrations.

This folder contains:
- example plugins
- reference implementations
- links to external plugin repositories
